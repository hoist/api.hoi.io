URL Root

All urls have the root of https://api.hoi.io

Authentication

Hoist uses a token based authentication. All calls should be made with an Authorization header set to Hoist where API Key can be found in the Hoist portal at https://portal.hoist.io

Event Stream 

A streaming API to get all events as they fire

Retrieve Event Stream
GET
/events?token={token}&filterBy={filterBy}&filterValue={filterValue}&timeoutMs={timeoutMs}

Retrieves a stream of events and sets up a stream token.

  • Initial requests can leave off the token and one will be generated for you. \

  • Subsequent requests with the token will return events since the token was created, a new token will be generated to continue the stream for future requests.

  • The token value will remain valid for 10 minutes after the token is created.

eg.

GET /events
Host: https://api.hoi.io
Content-Type: application/json

{
    "token":"abcdefg1234"
    "events":[{
        "eventName":"my-event"
    }]
}

then

GET /events?token=abcdefg1234
Host: https://api.hoi.io
Content-Type: application/json

{
    "token":"<newToken>",
    "events":[{
        ... any events since last poll using the id
        }]
}
  • Parameters
  • token
    string (optional) Example: abc123

    the token returned in the last request to continue where it left off

    filterBy
    'eventName' or 'correlationId' (optional) Example: eventName

    the field to filter the stream by

    filterValue
    string (optional) Example: eventid

    the value to filter the stream by

    timeoutMs
    number (optional) Example: 20000

    the number of ms to timeout when no events are available (will return an empty array) defaults to 10000
